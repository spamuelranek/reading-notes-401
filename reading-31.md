## [Javascript Review](https://www.taniarascia.com/es6-syntax-and-feature-overview/)
- variables
  - keywords:
    - var : global scope
    - let : scoped
    - const : scoped

- functions
  - `function func(a){}` : function declaration
  - `var func = function(a){}` : function expression
- arrow functions
  - `let func = (a) => {}`
    - does not have their own `this`

- Array iteration
```javascript
for (let i of arr){
  console.log(i)
}
```
